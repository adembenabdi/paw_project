# IoT RFID Access Control System

A web-based RFID door access control system built with PHP, MySQL, and ESP32.

## Author

**Adem Benabdi** - [GitHub](https://github.com/adembenabdi)

## Features

- User management with RFID card registration
- Door access control and monitoring
- Real-time dashboard with statistics
- Security alerts for failed access attempts
- Access logs and history tracking
- Admin panel with profile management

## Tech Stack

- **Backend:** PHP 8.0, MySQL 8.0
- **Frontend:** Bootstrap 5, Chart.js, Font Awesome
- **Hardware:** ESP32, MFRC522 RFID Reader
- **Deployment:** Docker, Docker Compose

## Quick Start

```bash
docker-compose up -d
```

Access the application at `http://localhost:8002`
